## Reporting a Vulnerability

We take the security of **MarkDiffusion** very seriously. If you discover a security vulnerability, please **DO NOT** open a public issue.

Instead, please report it privately by emailing the maintainers:

* **Primary Contact:** `panly24@mails.tsinghua.edu.cn`
* **Secondary Contact:** `sily23@mails.tsinghua.edu.cn`

Please include as much information as possible (e.g., steps to reproduce, environment details) to help us verify and fix the issue. We will do our best to respond to your report as soon as possible.
